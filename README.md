# Simple Todo (Vite + React)

This is a minimal frontend-only todo app built with Vite and React.

Install and run:

```bash
npm install
npm run dev
```

Open http://localhost:5173 (or the URL printed by Vite).

Features:
- Add todos
- Toggle complete
- Delete todos
- Persistence in `localStorage`

Files of interest:
- `src/App.jsx` — main app and add form
- `src/components/TodoList.jsx` — list view
- `src/components/TodoItem.jsx` — individual todo
- `src/styles.css` — simple styles

Notes:
- This is frontend-only; no backend required.
- To build for production: `npm run build` and `npm run preview` to serve the built site.
